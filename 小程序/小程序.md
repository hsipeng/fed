# 小程序



##小程序上线流程



1. 注册账号和小程序

​	

https://mp.weixin.qq.com/

「设置」->「开发设置」获取到「AppID」

2. 安装微信开发者工具

https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html



3. 开发者工具上传代码



4. 提交审核



微信 小程序助手 审核

网页提交审核

一版截图提交就好



5. 审核之后发布



微信 小程序助手 发布

网页提交发布
